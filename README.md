# 🧱 Neo-Brutalism Portfolio - Gautham Krishna B

A bold, raw, and unapologetic cybersecurity portfolio design featuring hard edges, strong shadows, and high contrast.

## 🎨 Design Philosophy

Neo-Brutalism embraces the raw, unpolished aesthetic of brutalism with modern web design principles. This portfolio uses:
- **Hard borders** (3px solid black)
- **No rounded corners** (or minimal)
- **Bright yellow accent** (#FFD166) for contrast
- **Monospace typography** for technical feel
- **Aggressive shadows** with no blur
